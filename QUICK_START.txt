FASTEST WAY TO GET STARTED (2 STEPS)

Step 1️⃣  Run Diagnostics
  📁 Go to c:\codes folder
  🖱️  Double-click: diagnose.bat
  ⏳ Wait for it to start the server

Step 2️⃣  Open in Browser
  🌐 Open your browser
  🔗 Go to: http://localhost:3000
  ⏱️  Wait 5-10 seconds for page to load

─────────────────────────────────────
  1. Open PowerShell/Command Prompt

Option B - With Python
──────────────────────

  2. Go to folder: cd c:\codes
  3. Start server: python server.py
  4. Open browser: http://localhost:3000

COMMON ISSUES & FIXES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "Page takes too long to load"
   ✅ Open browser IMMEDIATELY after server starts
   ✅ Give it 10 seconds minimum
   ✅ Check internet connection

❌ "Cannot reach / Connection refused"
   ✅ Make sure server is running (check terminal)
   ✅ Try: http://localhost:3000
   ✅ Check Windows Firewall (allow Node.js/Python)

❌ "Port 3000 is already in use"
   ✅ Close other apps using port 3000
   ✅ Or edit server.js line 6: const PORT = 3001;

❌ "File not found"
   ✅ Make sure nutriplan.html is in c:\codes
   ✅ Check spelling: nutriplan (not nutriplan)

❌ "Cannot find node/python"
   ✅ Download Node.js from: https://nodejs.org
   ✅ Or Python from: https://python.org
